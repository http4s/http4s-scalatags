# http4s-scalatags

Provides http4s entity codec instances for [scalatags][scalatags].

## SBT coordinates

```scala
libraryDependencies ++= Seq(
  "org.http4s" %% "http4s-scalatags" % http4sScalatagsV
)
```

## Compatibility
